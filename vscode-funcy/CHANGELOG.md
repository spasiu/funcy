# Change Log

All notable changes to the "funcy-lang" extension will be documented in this file.

## [0.1.0] - 2025-12-07

### Added
- Initial release
- Syntax highlighting for Funcy language
- Support for variables, functions, module exports
- Support for event subscriptions and triggers
- Support for await operator
- Support for built-in functions and modules
- Support for type annotations
- Comment, bracket, and auto-closing pair configuration
